Podemos también derivar una clase especializada desde una general:
class libro {
nombre,
props
...
mets
}
class libroAntiguo : libro {
edad,
añadirEdad = function (age) {
edad = age;
return this;
}
} // los objetos de esta clase tendrán nombre & edad
JavaScript usa class nuevaClase extends claseBase en vez del operador :.
Para efectos prácticos, esto es como copiar y añadirle a una clase, pero con un tanto más facilidad.

Además, podemos crear objetos del nuevo tipo con el nombre de su "clase progenitora", solo necesitamos llamar al nuevo constructor:
libro libroDeMilSeiscientos = new libroAntiguo();
libro.añadirEdad(423); // libroDeMilSeiscientos.edad, si configuramos el setter bien, va a ser igual a 423.
Se dice que la nueva clase "hereda" las propiedades de la anterior, o que la nueva clase es "hija" de la base.

También podemos re-definir métodos para nuestras nuevas clases, a lo que llamamos polimorfismo:
clase libro {
...
añadirNombre = function (nom) {
nombre = nom;
return this;
}
}
clase libroAntiguo {
...
añadirNombre = function (nom) {
nombre = nom + " — Antiguedad";
return this;
}
}
libro libroDeMilSeiscientos = new libroAntiguo();
Write(libroDeMilSeiscientos.añadirNombre("Las Mil y Una Noches, Edición Medieval").nombre); // imprimirá "Las Mil y Una Noches, Edición Medieval — Antiguedad"
